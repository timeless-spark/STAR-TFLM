# STAR-TFLM

This repository contains the source code for the paper "Accelerating Mixed-Precision QNN Inference on RISC-V MCUs with the STAR-MAC unit". The paper is under review process for pubblication in the IEEE ACCESS. Preprint version is available in the TechRxiv [https://doi.org/10.36227/techrxiv.173161032.20267860/v3](https://doi.org/10.36227/techrxiv.173161032.20267860/v3)
