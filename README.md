# STAR-TFLM

This repository contains the source code for the paper "An End-to-End Flow to Deploy and Accelerate TinyML Mixed-Precision Models on RISC-V MCUs". The paper is under review process for pubblication in the IEEE Transactions on Computer-Aided Design of Integrated Circuits and Systems (TCAD). Preprint version is available in the TechRxiv [https://doi.org/10.36227/techrxiv.173161032.20267860/v2](https://doi.org/10.36227/techrxiv.173161032.20267860/v2)
